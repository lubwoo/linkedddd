Welcome — Quick site linking demo

This small site shows an internal link (About) and an external link (Example.com). The external link opens in a new tab for safety and better UX.

Go to About Open Example.com

Tip: change the external URL to any website you need. Keep rel="noopener noreferrer" and target="_blank" for safety.

Why use external links like this?

  • Keeps users on your site while allowing them to visit resources in a new tab.
  • Using rel="noopener noreferrer" improves security and performance.

Need more pages?

Copy the pattern used above to create extra pages (contact.html, products.html) and link them from the nav or from buttons.
